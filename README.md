# Svelte-Tailwind-JIT-Vite

Quickly scaffold a Svelte App with Just-in-time TailwindCSS compiler on Vite.

# Install

```
git clone https://github.com/reepolee/svelte-tailwind-jit-vite.git
cd svelte-tailwind-jit-vite
npm i
npm run dev -- --open
```
